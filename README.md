# Shortest Path Navigation System
The Shortest Path Navigation System is a project based on graph algorithms and data structures that finds the shortest path between different locations or nodes in a network. 
The project uses Dijkstra’s Algorithm to calculate the minimum distance path efficiently. This project demonstrates the practical application of graphs, priority queues, and 
path optimization techniques used in real-world navigation systems such as GPS and route planning applications.

## Features
- Shortest path calculation
- Graph-based implementation
- Efficient route optimization

## Technologies Used
- C / C++
- Data Structures & Algorithms

## Concepts Used
- Graphs
- Dijkstra’s Algorithm
- Priority Queue

## Applications
- GPS Navigation
- Route Optimization
